# RISC-Pipeline-Processor
5-stage RISC Pipeline Processor designed using Verilog HDL and simulated in Xilinx Vivado.
## Modules Implemented
- ALU
- Register File
- Instruction Memory
- Control Unit
- Instruction Fetch (IF)
- IF/ID Pipeline Register
- Top Module

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Features
- Instruction Fetch
- Instruction Decode
- Arithmetic and Logic Operations
- Register Read/Write Operations
